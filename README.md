# Pricing Panel
Pricing panel layout for three different payment tiers.

## Features
* Flexbox
* media queries
* responsive design
* CSS reset
  * compressed set of CSS rules that *resets* the styling of all HTML elements to a consistent baseline
* Google Fonts API
